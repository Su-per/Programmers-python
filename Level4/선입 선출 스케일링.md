## 풀이
```python
def solution(n, cores):
    remain = [0 for _ in cores]
    while n:
        for i in range(len(cores)):
            if remain[i] == 0:
                n-=1
                if n == 0: return i+1
                remain[i] += cores[i]
        remain = list(map(lambda x: x-1, remain))
```
