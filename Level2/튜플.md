## 코드
```python
def solution(s):
    answer = []
    s = eval(s.replace("{", '[').replace("}", "]"))
    s.sort(key = lambda x: len(x))
    for i in s:
        for j in i:
            if not j in answer:
                answer.append(j)
    return answer
```
