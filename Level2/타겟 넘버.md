## 코드

```python
def solution(numbers, target):
    num = [0]
    def recursion(index, hap):
        if index == len(numbers):
            if hap == target: 
                num[0] += 1
            return
        recursion(index+1, hap + numbers[index])
        recursion(index+1, hap - numbers[index])
        
    recursion(0, 0)
    return num[0]
```
