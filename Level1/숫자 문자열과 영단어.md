## 코드
```
def solution(s):
    eng = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"]
    for i in range(len(eng)):
        s = s.replace(eng[i], str(range(0,10)[i]))
    return int(s)
```
