## 코드
```python
from itertools import combinations
def is_prime(num):
    for i in range(2, num):
        if num % i == 0:
            return False
    return True
def solution(nums):
    return len([i+j+k for i,j,k in list(combinations(nums, 3)) if is_prime(i+j+k)])
```
