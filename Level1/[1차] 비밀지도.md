## 코드
```python
def solution(n, arr1, arr2):
    res = [[0 for i in range(n)] for j in range(n)]
    
    for i in range(n):
        b1 = format(arr1[i], 'b').zfill(n)
        b2 = format(arr2[i], 'b').zfill(n)
        for j in range(n):
            res[i][j] = "#" if int(b1[j]) | int(b2[j]) else " "
    return list(map(''.join, res))
```
